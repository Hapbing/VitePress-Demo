## ![我的头像](/image/logo.png){.avatar}桌面端合并工具



>基于 Tauri 和 Vue 3 开发的桌面应用程序，用于将多张照片合并为 PDF 文档。

## 技术栈

- **前端框架**：Vue 3 + Vite
- **UI 组件库**：Element Plus
- **桌面应用框架**：Tauri 2.0
- **PDF 处理**：jsPDF
- **看板**：OhMyLive2D
